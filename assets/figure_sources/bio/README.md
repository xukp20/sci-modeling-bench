# Biomolecular overview figure sources

The published figure is `assets/sci-modeling-bench-bio-overview.svg`. It is a
self-contained SVG built from the editable layout in this directory by
`build_bio_overview.py`.

## Included scientific imagery

- **Pho4-DNA complex:** RCSB PDB entry [1A0A](https://www.rcsb.org/structure/1A0A),
  assembly image downloaded from the RCSB structure summary page. RCSB makes
  these molecular images available under CC0 and encourages attribution.
- **GFP structure:** RCSB PDB entry [1GFL](https://www.rcsb.org/structure/1GFL),
  assembly image downloaded from the RCSB structure summary page, under the
  same RCSB CC0 image policy.
- **Ribosome translation illustration:** adapted from
  [Ribosome mRNA translation en.svg](https://commons.wikimedia.org/wiki/File:Ribosome_mRNA_translation_en.svg)
  by LadyofHats, released into the public domain. The adaptation removes the
  explanatory labels, leader lines, and white background, then crops and fades
  the lower edge for use in the montage.

RCSB usage policy: <https://www.rcsb.org/pages/usage-policy>

## Original task-specific vector layers

The 8-mer sequence landscape, 50-nt 5′UTR construct, fixed leader, uAUG,
Kozak/start-codon boundary, fixed eGFP reporter, polysome cue, and GFP variant
annotation were drawn specifically for SciModelingBench. They are schematic
visual encodings of the implemented task definitions, not additional measured
data plots.
