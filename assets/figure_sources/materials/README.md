# Materials-discovery overview figure sources

The published figure is `assets/sci-modeling-bench-materials-overview.svg`.
Run `build_materials_overview_v3_rect.py` to regenerate the final composition,
which uses rectangular photographic crops. The earlier builders and layouts are
retained as editable construction history, but their versioned exports are not
kept in the public `assets/` directory.

## Included imagery

- **YBCO lattice:** `Ybco002.svg` by Rswarbrick, Wikimedia Commons,
  CC BY-SA 3.0 / GFDL.
- **Meissner-effect photograph:** magnet levitating over a cooled
  superconductor, Mai-Linh Doan / David Monniaux, Wikimedia Commons,
  CC BY-SA 3.0 and compatible earlier licenses.
- **Sprague-Dawley rat photograph:** Jean-Etienne Minh-Duy Poirrier,
  Wikimedia Commons, CC BY-SA 2.0.
- **Molecular structures:** generated with RDKit from acetaminophen,
  ibuprofen, and caffeine SMILES present in the official CEBS DrugMatrix
  curated-chemicals table.

The elemental-composition strips, transition-temperature ranking, and treatment
arrow are original schematic vector layers. They identify the modeled objects
without presenting synthetic measurements or proxy score plots.
